Button — Tokens de Componente

Documentação completa dos design tokens do componente Button do DS Essential 2.0. Cada slot visual tem um token semântico rastreável até o primitivo correspondente.

4Variantes
3Tamanhos
6Estados
~48Tokens totais
Coleção de consumo: Os tokens de componente vivem na coleção Channel, grupo component. Nunca referenciam Primitive diretamente — sempre via Theme ou Brand.
Anatomia

Anatomia do Componente

O Button é composto por até 3 slots: ícone à esquerda (opcional), label e ícone à direita (opcional). Cada slot possui seus próprios tokens de cor e dimensão.

Primary · LG
Primary · MD
Primary · SM
Secondary · MD
Ghost · MD
Destructive · MD
Disabled · MD

Slots do componente

icon-leading
Ícone à esquerda do label. Opcional. Herda a cor do foreground da variante atual.
label
Texto do botão. Sempre obrigatório. Verbo no infinitivo + substantivo específico.
icon-trailing
Ícone à direita do label. Opcional. Usado para indicar ação de expansão ou link externo.
background
Superfície do botão. Pode ser opaco, transparente ou com borda (secondary).
border
Contorno do botão. Presente apenas em Secondary (1px) e no anel de foco (2px, todos).
focus-ring
Anel de foco externo. 2px, offset de 2px. Obrigatório para WCAG 2.4.7 (contraste mínimo 3:1).

Forma & Borda

Forma e Borda

O Button usa um único valor de raio para todos os tamanhos e variantes. O anel de foco é sempre 2px de espessura com offset de 2px do elemento.

// Button.tsx — consumindo os tokens
import { buttonTokens as t } from "@ps/design-tokens/button"

type Variant = "primary" | "secondary" | "ghost" | "destructive"
type Size    = "sm" | "md" | "lg"

const variantStyles: Record<Variant, React.CSSProperties> = {
  primary: {
    background:  t.color.bg.primary,
    color:       t.color.fg.primary,
  },
  secondary: {
    background:  t.color.bg.secondary,
    color:       t.color.fg.secondary,
    border:      `${t.numericValues.borderWidthSecondary} solid ${t.color.border.secondary}`,
  },
  ghost: {
    background:  t.color.bg.ghost,
    color:       t.color.fg.ghost,
    border:      "none",
  },
  destructive: {
    background:  t.color.bg.destructive,
    color:       t.color.fg.destructive,
  },
}

export function Button({
  variant = "primary",
  size    = "md",
  disabled,
  children,
  ...props
}: ButtonProps) {
  const sz = t.numericValues[size]
  return (
    <button
      style={{
        ...variantStyles[variant],
        borderRadius:  t.numericValues.radius,
        height:        sz.height,
        padding:       `${sz.paddingY} ${sz.paddingX}`,
        gap:           sz.gap,
        fontSize:      t.typography[size].fontSize,
        fontWeight:    t.typography.fontWeight,
        display:       "inline-flex",
        alignItems:    "center",
        opacity:       disabled ? 0.5 : 1,
        cursor:        disabled ? "not-allowed" : "pointer",
      }}
      disabled={disabled}
      {...props}
    >
      {children}
    </button>
  )
}

import UIKit

public enum ButtonTokens {

  public enum Color {
    public enum Background {
      public static let primary         = UIColor(hex: "#0047CC")
      public static let primaryHover     = UIColor(hex: "#003087")
      public static let primaryActive    = UIColor(hex: "#001A5C")
      public static let primaryDisabled  = UIColor(hex: "#D8D8EE")
      public static let destructive      = UIColor(hex: "#CC2200")
      public static let destructiveHover = UIColor(hex: "#7A1300")
    }
    public enum Foreground {
      public static let primary     = UIColor.white
      public static let secondary   = UIColor(hex: "#0047CC")
      public static let ghost       = UIColor(hex: "#0047CC")
      public static let disabled    = UIColor(hex: "#8888AA")
      public static let destructive = UIColor.white
    }
    public enum Border {
      public static let secondary         = UIColor(hex: "#0047CC")
      public static let secondaryDisabled  = UIColor(hex: "#D8D8EE")
      public static let focusRing         = UIColor(hex: "#5B9BFF")
    }
  }

  public enum Size {
    public enum SM {
      public static let height:   CGFloat = 32
      public static let paddingX: CGFloat = 12
      public static let paddingY: CGFloat = 6
      public static let iconSize: CGFloat = 16
      public static let gap:      CGFloat = 4
    }
    public enum MD {
      public static let height:   CGFloat = 40
      public static let paddingX: CGFloat = 16
      public static let paddingY: CGFloat = 9
      public static let iconSize: CGFloat = 20
      public static let gap:      CGFloat = 4
    }
    public enum LG {
      public static let height:   CGFloat = 48
      public static let paddingX: CGFloat = 20
      public static let paddingY: CGFloat = 12
      public static let iconSize: CGFloat = 24
      public static let gap:      CGFloat = 8
    }
  }

  public static let radius:         CGFloat = 8
  public static let borderWidth:    CGFloat = 1
  public static let focusRingWidth: CGFloat = 2
  public static let fontSizeSm:     CGFloat = 12
  public static let fontSizeMd:     CGFloat = 14
  public static let fontSizeLg:     CGFloat = 16
