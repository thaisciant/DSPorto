# Contract: Button iOS Component API (Swift / SwiftUI)

**Platform**: iOS 16+ — SwiftUI + Swift 5.10
**Package**: `PSButton` (Swift Package Manager)
**Date**: 2026-06-15

---

## Public API

### Component signature

```swift
PSButton(
    label: "Publicar bundle",
    variant: .primary,           // PSButtonVariant — optional, default .primary
    size: .md,                   // PSButtonSize — optional, default .md
    isDisabled: false,           // Bool — optional, default false
    isLoading: false,            // Bool — optional, default false
    fullWidth: false,            // Bool — optional, default false
    iconLeading: Image(...),     // Image? — optional
    iconTrailing: nil,           // Image? — optional
    accessibilityLabel: nil,     // String? — optional, defaults to label
    action: { /* callback */ }   // () -> Void — required
)
```

### Types

```swift
public enum PSButtonVariant {
    case primary
    case secondary
    case ghost
    case destructive
}

public enum PSButtonSize {
    case sm   // height: 32, paddingX: 12, paddingY: 6,  iconSize: 16, gap: 4, fontSize: 12
    case md   // height: 40, paddingX: 16, paddingY: 9,  iconSize: 20, gap: 4, fontSize: 14
    case lg   // height: 48, paddingX: 20, paddingY: 12, iconSize: 24, gap: 8, fontSize: 16
}
```

### View struct (public surface)

```swift
public struct PSButton: View {
    public let label: String          // required; non-empty
    public let variant: PSButtonVariant   // default .primary
    public let size: PSButtonSize         // default .md
    public let isDisabled: Bool           // default false
    public let isLoading: Bool            // default false; implies isDisabled
    public let fullWidth: Bool            // default false
    public let iconLeading: Image?        // optional
    public let iconTrailing: Image?       // optional
    public let accessibilityLabel: String? // nil → uses label
    public let action: () -> Void         // required

    public var body: some View { ... }
}
```

### ViewModel (ObservableObject)

```swift
// Internal — not part of public API; consumed only by PSButton.body
final class PSButtonViewModel: ObservableObject {
    @Published private(set) var state: PSButtonState = .default

    func resolveTokenSet(variant: PSButtonVariant, state: PSButtonState) -> PSButtonTokenSet
    func handlePress()
    func handleHover(isHovering: Bool) // macOS Catalyst / iPadOS pointer
}
```

## Token constants consumed

All values from `PSButtonTokens.swift` — generated by Style Dictionary.
No UIColor or CGFloat literals in `PSButton.swift` or `PSButtonViewModel.swift`.

```swift
ButtonTokens.Color.Background.primary        // UIColor(hex: "#0047CC")
ButtonTokens.Color.Background.primaryDisabled // UIColor(hex: "#D8D8EE")
ButtonTokens.Color.Foreground.primary        // UIColor.white
ButtonTokens.Color.Border.focusRing          // UIColor(hex: "#5B9BFF")
ButtonTokens.Size.MD.height                  // CGFloat: 40
ButtonTokens.Size.MD.paddingX                // CGFloat: 16
ButtonTokens.radius                          // CGFloat: 8
ButtonTokens.fontSizeMd                      // CGFloat: 14
ButtonTokens.fontWeight                      // Font.Weight.semibold
```

## Accessibility contract

| Modifier | Value | When |
|----------|-------|------|
| `.accessibilityLabel` | `accessibilityLabel ?? label` | Always |
| `.accessibilityAddTraits(.isButton)` | — | Always |
| `.accessibilityAddTraits(.isStaticText)` | — | Never — button is interactive |
| Disabled via `.disabled(isDisabled \|\| isLoading)` | — | When disabled/loading |
| `accessibilityValue("Loading")` | — | When `isLoading = true` |
| Focus ring | 2 pt border in `ButtonTokens.Color.Border.focusRing` | On keyboard focus (iPadOS / macOS) |

## Invariants (test-verifiable)

1. `label` MUST NOT be empty — `preconditionFailure` in debug builds.
2. Tapping a disabled or loading button MUST NOT call `action`.
3. All colour values MUST come from `PSButtonTokens` — no UIColor literals in source.
4. `isLoading = true` implicitly sets `isDisabled = true` in the ViewModel.
5. Size tokens resolve to CGFloat values matching the data model table exactly.
